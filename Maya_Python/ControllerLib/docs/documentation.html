"Controller Library" Documentation

-------------------------------------------------------------------------------------------------------------------------------------

Copyright (c) Junru Tao 2021

This is a Maya Python library for managing controllers and procedural rigging utilities. This script is based on Dhruv's tutorial
(link) and made adaptations, organisations and new functionalities into his origninal scripts. Also check out Dhruv's
GitHub page for more information and learning materials.

1.0 Installation

1. Place the entire folder ControllerLib into maya's default user's script path /{$User}/maya/scripts/

2.1 Loading the script in maya either through the default shelf tool under /{$package}/maya/{$MayaVersion}/prefs/shelves/{FILE}.mel
under the same maya user's folder file structure.

2.2 or simply use the code here in Maya' script editor
from ControllerLib import libraryUI as LU
reload(LU)
ui = LU.open_editor()
for example. To load just the `Controller Manager` library

controller_manager(CM).py - API Guide if without UI

   ⚫ [Function] CM.create_controller_folder(directory='', o=False)

❖ Create a new Controller Folder unless the directory already exists if directory not specified,{$MAYA_USER_DIR}/{$CONTROLLER_FOLDER} will be set as default.
❖ Use o=True flag to open up the controller folder in file explorer

❖ Parameters ❖ Type ❖ Input Description
directory string A directory to specify, non will create
o boolean if Open this directory in file explorer
➥ RETURN -> - -

   ⬜ [Class] CM.ControllerManager(dict)

❖ [Controller Manager]

- Dictionary Class handling create, save and loading controller geometries and its adjacent information.
-> geo.ma [maya ASCII, controller geometry]
-> geo.json [meta data, controller info]
-> geo.jpg [icon-thumbnail]

❖ ❖ Functions
- ControllerManager.save(...)
- ControllerManager.find(...)
- ControllerManager.load(...)

+ + + + + + + + + + + +
